Opportunity Mapper

JP Carrascal (2021)
Web app to calculate Opportunity Scores and Map
Opportunity Mapper is a web application designed to calculate and visualize Opportunity Scores across different geographical regions or datasets. This tool provides researchers, analysts, and decision-makers with an interactive platform to assess and map various opportunities based on quantitative metrics and data analysis.

The application features an intuitive interface that allows users to input data parameters, calculate opportunity scores using customizable algorithms, and generate interactive maps that visualize the results. This makes it easier to identify patterns, trends, and areas of high opportunity across different contexts.

Built with modern web technologies, Opportunity Mapper serves as both a practical tool for data analysis and a demonstration of interactive data visualization techniques in web-based applications.

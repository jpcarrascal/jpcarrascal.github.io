Opportunity Mapper

JP Carrascal (2021)
Web app to calculate Opportunity Scores and Map
The Opportunity Score and the Opportunity Map (or Landscape) is a tool for exploring a user problem space by analysing the gap between the importance of a need and the satisfaction with current solutions. It was introduced by Anthony Ulwick as part of his Outcome-Driven Innovation framework.

Opportunity Mapper is a web application that helps calculating the Opp Score and visualising the data in a map. This tool provides researchers, analysts, and decision-makers with an interactive platform to assess and map various opportunities and prioritize them based on quantitative metrics and data analysis.

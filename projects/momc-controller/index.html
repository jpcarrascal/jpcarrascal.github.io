MoMc Modular MIDI Controller

JP Carrascal (2021)
Arduino-based modular MIDI pedal controller
MoMc (Modular MIDI Controller) is an Arduino-based modular MIDI pedal controller designed for musicians who need flexible, customizable control solutions for their electronic instruments and effects. This open-source project provides a robust foundation for creating personalized MIDI control interfaces.

The modular design philosophy allows users to configure the controller according to their specific needs, whether for live performance, studio work, or experimental music creation. Built on the Arduino platform, MoMc leverages the accessibility and flexibility of open-source hardware to democratize advanced MIDI control technology.

The project includes comprehensive documentation and code examples, making it accessible to both experienced developers and musicians interested in learning about hardware-software integration in musical applications.

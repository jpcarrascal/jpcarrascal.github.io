Lockdown

JP Carrascal (2020)
An interactive videoclip for Spacebarman's song 'Lockdown'. Built with WebVR/WebXR, best experienced with an Oculus Quest.
Lockdown is an innovative interactive videoclip created for Spacebarman's song 'Lockdown', developed during the global pandemic of 2020. This project represents a pioneering approach to music visualization, combining virtual reality technology with artistic expression to create an immersive audiovisual experience.

Built using WebVR/WebXR technologies, the project is accessible through web browsers while delivering a fully immersive experience, particularly when viewed with VR headsets like the Oculus Quest. The browser-based approach ensures accessibility while pushing the boundaries of what's possible with web-based virtual reality.

The interactive videoclip reflects the zeitgeist of 2020, exploring themes of isolation and connection through virtual spaces. This work demonstrates the potential of VR as a medium for artistic expression and musical storytelling, offering viewers an active role in the narrative experience.
